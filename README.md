# Hospital-Efficiency-Analysis
This project focuses on analyzing hospital operational workflows to identify inefficiencies contributing to increased patient wait times and resource underutilization.

The objective was to use business analysis and data analytics techniques to improve operational efficiency, optimize resource allocation, and enhance patient experience through data-driven recommendations.

# Business-Problem

Hospitals often face operational bottlenecks that lead to:
- Increased patient wait times
- Resource allocation inefficiencies
- Delays in treatment and consultation
- Reduced patient satisfaction

This project aims to identify process inefficiencies and recommend operational improvements using data analysis and dashboarding techniques.

# Business-Objective

- Reduce patient wait times
- Improve resource utilization
- Identify operational bottlenecks
- Enhance workflow efficiency
- Support data-driven decision-making
- Improve service delivery performance
